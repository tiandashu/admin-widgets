## start
基于@vue/cli 开发

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
### 打包 lib
```
npm run lib
```

### 结构
- examples 开发示例
- packages 组件源码
